# NCS Lens 修正・改善リスト

修正指示や気づきをここに溜めて、まとめて対応する。

---

## 対応済み

- [x] チャット内の名前を「さん」付けにする（`{childName}` → `なぎささん`）
- [x] 結果画面の「くん」→「さん」に統一
- [x] 回答は戻れるようにする
- [x] 1問目を「セルフコントロール（感情を表現したり我慢できたりした）」の選択肢に変更する
- [x] チャットは積み上げ形式ではなく、1問につき1つの回答窓にする（前の設問は戻って編集できるようにする）
- [x] チャットには選択式の回答結果も表示する
- [x] 「夢中になっていること」に音楽を追加する（「つくる・描く」→「表現する系（工作、絵、ブロック、音楽など）」に変更）
- [x] 「うまくいかないときの反応」についての設問を、困りごとの設問の後に移動する
- [x] 設問分析のローディング時間が長過ぎる → クライアント側に55秒タイムアウト追加、エラー表示・リトライ機能追加
